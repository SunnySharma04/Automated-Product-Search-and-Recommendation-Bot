# Automated Product Search and Recommendation Bot

An Intelligent Process Automation (IPA) solution developed using **UiPath** during the **Capgemini Intelligent Process Automation (IPA) Training**.

## Overview

This project automates the process of searching products across multiple e-commerce websites, extracting product information, comparing products based on predefined criteria, and generating a recommendation report. The automation minimizes manual effort and helps users make informed purchasing decisions.

## Key Features

- Automated product search
- Web automation and data extraction
- Product comparison
- Excel report generation
- Exception handling
- Modular workflow design

## Technologies Used

- UiPath
- Web Automation
- Data Scraping
- Excel Automation
- UI Selectors
- Exception Handling

## Project Workflow

The automation performs the following steps:

1. Accepts the product search input.
2. Searches products on supported e-commerce websites.
3. Extracts product details such as name, price, ratings, and availability.
4. Compares products based on predefined criteria.
5. Generates a recommendation report in Excel.
6. Handles exceptions and completes the automation.
